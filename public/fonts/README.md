# Font Files

Please add the following font files to this directory:

1. **CrimsonText-SemiBold.ttf** - Crimson Text, weight 600, normal
2. **CrimsonText-SemiBoldItalic.ttf** - Crimson Text, weight 600, italic
3. **AveriaSerifLibre-Bold.ttf** - Averia Serif Libre, weight 700, normal
4. **AveriaSerifLibre-BoldItalic.ttf** - Averia Serif Libre, weight 700, italic
5. **athena-unicode.ttf** - New Athena Unicode, weight 400, normal

These fonts are required for the poster generator to work correctly.

## Font Sources

- Crimson Text: https://fonts.google.com/specimen/Crimson+Text
- Averia Serif Libre: https://fonts.google.com/specimen/Averia+Serif+Libre
- New Athena Unicode: http://apagreekkeys.org/NAUdownload.html
